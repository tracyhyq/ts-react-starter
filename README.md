# ts-react-starter

ts + react 前端项目脚手架

## 使用技术栈:

* typescript 3.x
* koa
* postcss
* webpack4.x
* react16.x
* mobx
* antd3.x

## 开发环境下运行项目

推荐使用 npm 安装:

```bash
$ npm i
$ npm run start:client
$ npm run start:server
```

访问：http://localhost:3001/#/list
